# 📊 Análisis de Ventas - Alura Store Latam

## 📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar los datos de ventas de cuatro tiendas de **Alura Store** con el fin de determinar cuál de ellas debería venderse, basándose en diferentes métricas de desempeño.

El análisis se realizó utilizando Python en un notebook de **Google Colab**, aplicando técnicas básicas de análisis de datos y visualización.

---

## 🎯 Objetivo

Identificar la tienda con menor rendimiento para recomendar su venta y así optimizar la estrategia comercial de la empresa.

---

## 🛠 Tecnologías utilizadas

* Python
* Pandas
* Matplotlib
* Google Colab

---

## 📂 Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

* Facturación total por tienda
* Categorías de productos más vendidas
* Productos más vendidos y menos vendidos
* Calificación promedio de cada tienda
* Costo promedio de envío por tienda

Los resultados fueron visualizados mediante gráficos de barras para facilitar la comparación entre tiendas.

---

## 📊 Resultados

Después de analizar los datos, se observó que **la Tienda 4 presenta el peor desempeño general**, ya que:

* Tiene la **menor facturación total**
* Presenta **menor volumen de ventas**
* No muestra ventajas significativas en **calificación de clientes**
* Sus costos de envío no compensan su bajo rendimiento

---

## 🧠 Conclusión

Se recomienda **vender la Tienda 4**, ya que su rendimiento es inferior al de las demás tiendas analizadas. Esto permitiría concentrar recursos y esfuerzos en las tiendas con mejor desempeño y mayor rentabilidad.
